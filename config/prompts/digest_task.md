Сделай дайджест чата за период.

# Что тебе дают для каждого BURST

- **Пики плотности** — 10-минутные слоты с максимальным числом сообщений внутри окна.
- **Характерные слова (TF vs фон)** — слова, которые часто встречались именно в этом окне, но редко в остальном периоде.
- **Характерные пары слов (биграммы)** — то же, но для пар. Самый сильный сигнал темы: «соленая карамель», «молодежная столица», «филлер ботокс».
- **Топ обсуждаемых сообщений (root + ответы)** — реальные дискуссии. Прямой ответ на вопрос «о чём срач».
- **КАНДИДАТЫ В ЦИТАТЫ** — заранее отобранные содержательные сообщения. ЦИТИРОВАТЬ МОЖНО ТОЛЬКО ИХ.
- **Сэмпл сообщений окна** — содержательные сообщения (короткие реплики-междометия отфильтрованы).

# Алгоритм

Для каждого BURST:
1. Посмотри биграммы и характерные слова — обычно одна-две темы видны сразу.
2. Подтверди гипотезу темы через reply chains.
3. **Если в окне 2+ независимые темы** — назови все, не своди в одну. Перечисляй через `+`.

# Формат ответа

ВСПЛЕСКИ
Для каждого BURST одной-двумя строками:
- **[BURST N: дата, время МСК]** — конкретная тема, кто триггернул/доминировал.
- Если в одном BURST несколько тем — отдельные строки с тем же номером BURST.

ЦИТАТЫ
0–3 цитаты ТОЛЬКО из списка КАНДИДАТОВ. Формат: `[Qn.k] @user (HH:MM): текст`. Если кандидаты не подходят — секцию опусти.

ВАЙБ
Одна строка про общий тон периода.

# Запрещено

- Эмодзи в ответе. Заголовки секций — без эмодзи (только текст: «ВСПЛЕСКИ», «ЦИТАТЫ», «ВАЙБ»).
- Парафразы без сути («обсуждали разное», «ночной марафон», «офисные болтовни»).
- Цитировать что-либо кроме списка кандидатов.
- Выдумывать факты и события.
- Цензурировать мат/сленг.
- Вступительные общие фразы.
- Сводить разные темы окна в одну расплывчатую.

# Пустое окно

Если в окне биграммы пустые и нет чейнов: «BURST N: ровный трёп без явной темы, доминировали X и Y».

Длина: до 1500 символов.
