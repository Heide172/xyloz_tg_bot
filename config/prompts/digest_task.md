Сделай дайджест чата за период.

# Входные данные

Для каждого BURST даны: пики плотности, характерные слова и пары слов (TF vs фон), reply-цепочки (root + ответы), КАНДИДАТЫ В ЦИТАТЫ, сэмпл сообщений (короткие отфильтрованы).

# Алгоритм

Для каждого BURST: смотри биграммы и характерные слова → подтверди через reply chains. Если в окне 2+ независимые темы — назови все через `+`, не своди в одну.

# Формат

🔥 ВСПЛЕСКИ
• **[BURST N: дата, время МСК]** — конкретная тема, кто триггернул/доминировал.
• Если в одном BURST несколько тем — отдельные строки с тем же номером.

🎭 ЦИТАТЫ
0–3 цитаты ТОЛЬКО из списка КАНДИДАТОВ. Формат: `[Qn.k] @user (HH:MM): текст`. Если кандидаты не подходят — секцию опусти.

📈 ВАЙБ
Одна строка про общий тон периода.

# Запрещено

- Парафразы без сути («обсуждали разное», «ночной марафон», «офисные болтовни»).
- Цитировать что-либо кроме списка кандидатов.
- Выдумывать факты и события.
- Цензурировать мат/сленг.
- Вступительные общие фразы.
- Сводить разные темы окна в одну расплывчатую.

# Пустое окно

Если в окне биграммы пустые и нет чейнов: «BURST N: ровный трёп без явной темы, доминировали X и Y».

Длина: до 1500 символов.
